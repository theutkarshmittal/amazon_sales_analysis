# Amazon Sales Dashboard

## Project Overview

This project analyzes 128,975 Amazon sales records using Power BI to uncover business insights related to revenue, customer behavior, fulfilment performance, and order status.

## Dashboard Features

* Total Revenue Analysis
* Total Orders Analysis
* Average Order Value
* Cancellation Rate
* Revenue by State
* Revenue by Category
* Revenue by Fulfilment Method
* Top Cities by Revenue
* Order Status Distribution

## Data Cleaning

The dataset was cleaned using Power Query by:

* Removing unnecessary columns
* Handling missing values
* Standardizing text fields
* Correcting data types

## Key Insights

* Maharashtra generated the highest revenue.
* Bengaluru was the top revenue-generating city.
* Amazon fulfilment generated significantly more revenue than merchant fulfilment.
* The Set category contributed the highest revenue.
* Approximately 14% of orders were cancelled.
