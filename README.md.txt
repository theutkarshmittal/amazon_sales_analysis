# Amazon Sales Analysis

## Project Overview

This project analyzes 128,975 Amazon sales records using Python, Pandas, SQL, and data visualization.

## Tools Used

- Python
- Pandas
- Matplotlib
- MySQL

## Dataset Size

- Rows: 128,975
- Columns: 23

## Key Findings

- Set was the highest revenue-generating category.
- Maharashtra generated the highest revenue and order volume.
- Bengaluru was the top revenue-generating city.
- Amazon fulfilment generated more than twice the revenue of merchant fulfilment.
- More than 99% of revenue came from B2C customers.
- Approximately 14% of orders were cancelled.
